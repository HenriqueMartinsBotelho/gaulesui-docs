# Gaules UI

Gaules UI é uma biblioteca open source de componentes sofisticados e acessíveis para ajudar no desenvolvimento de novos recursos mais rapidamente.

- GitHub da lib: [**https://github.com/HenriqueMartinsBotelho/gaulesui_lib**](https://github.com/HenriqueMartinsBotelho/gaulesui_lib)
- Github desse site: [**https://github.com/HenriqueMartinsBotelho/gaulesui-docs**](https://github.com/HenriqueMartinsBotelho/gaulesui-docs)

## Tecnologias usadas

Além de React, Typescript são usados:

- **Storybook**: É uma ferramenta de código aberto para criar componentes de interface do usuário e páginas isoladamente.
  Ele simplifica o desenvolvimento, o teste e a documentação da interface do usuário.

- **Jest**: É uma estrutura de teste JavaScript construída em cima do Jasmine e mantida pela Meta.
  Ele foi projetado e construído por Christoph Nakazawa com foco na simplicidade e suporte para grandes aplicativos da web.
