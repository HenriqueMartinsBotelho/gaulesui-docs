# Button

Botões permitem que os usuários tomem ações e decisões com um simples toque.

## Exemplo de uso

```jsx highlight=2,6
import React from 'react'
import { Button } from "gaulesui_lib";

export const _app = () => {
  return (
    <Button label="Texto do botão" />;
  )
}
```
